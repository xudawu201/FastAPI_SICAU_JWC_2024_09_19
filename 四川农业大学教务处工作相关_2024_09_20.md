##### 遗传算法概念2024_09_23

遗传算法是一种模拟自然选择过程的优化方法，适用于求解复杂的组合优化问题，如排课问题。在排课问题中，需要在多个时间段和教室之间安排一系列课程，并尽量满足一系列约束条件（如避免冲突、教师可用性、教室容量等）。以下是遗传算法在排课问题中的基本工作流程：

###### 1.初始种群生成 (Initialization)

   - **个体表示 (Chromosome Representation)**: 在排课问题中，每个个体（染色体）表示一种排课方案。其中染色体上的每个基因元素表示某门课程在某个时间段和教室的安排。
   - **种群初始化 (Population Initialization)**: 生成一定数量的随机排课方案，作为初始种群。

###### **2.适应度评估 (Fitness Evaluation)**

   - 为每个个体计算一个适应度值（Fitness Value），表示该排课方案的优劣。适应度值的计算根据以下几个方面：
     - 是否存在时间冲突（如同一时间段安排了多门课程在同一个教室）。
     - 教师的排课均衡（如教师是否安排多次或未安排课程）。
     - 课程优先级（如优先级高的课程是否排课）。
     - 禁止排课时间（如课程不可以安排在某一时间段）。

###### 3.选择 (Selection)

   - 根据适应度值选择父代个体，优良个体被选择的概率更高。常见的选择方法包括轮盘赌选择 (Roulette Wheel Selection)、锦标赛选择 (Tournament Selection) 等。这里以适应度排名只选取适应度最高的两位作为父母

###### 4.交叉 (Crossover)

   - 从选择的父代个体中随机挑选两个进行交叉操作，生成新的后代个体（即新的排课方案）。常用的交叉方法有：
     - 单点交叉 (Single-Point Crossover)：在染色体的某一位置切分，交换两部分。
     - 多点交叉 (Multi-Point Crossover)：在多个位置切分，并交换切分的片段。

###### 5. 变异 (Mutation)
   - 对后代个体进行小概率的随机变异操作，以增加种群的多样性。对于排课问题，变异表现为随机调整某门课程的时间和教室安排。

###### 6. **新一代种群形成 (New Population)**
   - 由选择、交叉和变异操作生成的新个体和适应度最高的父代个体构成新一代种群。（为了维持种群质量，将前一代种群中最优的个体（精英保留）直接保留到新一代。）

###### 7. **终止条件 (Termination Condition)**
   - 算法通常在满足以下条件之一时终止：
     - 达到了最大迭代次数。
     - 适应度值没有显著提升（收敛）。
     - 找到了满足所有约束条件的最优排课方案（即适应度值达到设置的阈值）。

###### 8. **解的提取 (Solution Extraction)**
   - 从最终种群中选择适应度值最高的个体，作为最终的排课方案。

###### 优点
- **灵活性**：可以处理多种复杂的约束条件。
- **全局优化能力**：避免陷入局部最优解。
- **适应性强**：可以适用于不同规模和复杂度的排课问题。

###### 缺点
- **计算成本高**：对于大型问题，算法收敛速度较慢。
- **参数调节复杂**：种群大小、交叉概率、变异概率等参数的选择对算法性能影响较大。

###### 整体评价

遗传算法在排课问题中的应用具有较高的有效性和灵活性，实际实现中需要根据具体问题对适应度函数和参数进行调优。

##### 遗传算法具体实现2024_09_24

下面结合代码细节来介绍遗传算法的各个步骤。

###### 1. **课程和课程表的定义**

- **`Course`类**：定义课程的基本属性，包括名称、教师、持续时间、优先级和不可用时间段。
- **`Schedule`类**：维护一个课程表，包含课程、时间段和教室的信息。初始化时，所有时间段和教室组合都是空的。

###### 2. **适应度评估 (Fitness Function)**

- ```
  fitness
  ```

  方法通过以下方式评估一个排课方案的适应度：

  - 统计每门课程的安排次数，避免同一时间段内在多个教室安排同一课程（如有则降低适应度）。
  - 考虑课程的优先级，优先级高的课程得分更多。
  - 记录教师的安排次数，合理安排课程（如避免某位教师安排过多或过少课程，给予相应的适应度或降低适应度）。
  - 对于在不可用时间段安排的课程，减少较大的适应度。

###### 3. **变异 (Mutation)**

- `mutate`方法随机选择两个时间段和教室组合，交换其中的课程。增加种群基因的多样性，避免过早收敛到局部最优解。

###### 4. **交叉 (Crossover)**

- `crossover`方法从两个父代排课方案中生成两个子代。它随机选择一个交叉点，然后交换交叉点后的部分课程，生成新的排课方案。这有助于保留良好的排课特性，并组合不同个体的优点。

###### 5. **遗传算法的主循环**

- ```
  genetic_algorithm
  ```

  函数实现了遗传算法的主流程：

  - 初始化种群，随机生成一组排课方案。
  - 按照适应度对种群进行排序，保留适应度最高的个体。
  - 通过选择、交叉和变异生成新一代个体。
  - 每代结束时，输出当前最佳适应度和排课方案。
  - 如果达到预设的适应度阈值，则结束算法。

###### 6. **运行示例**

- 在`__main__`部分，定义了一组课程、时间段和教室，设置种群大小和适应度阈值，并运行遗传算法以寻找最佳排课方案。

import random

class Course:
    def __init__(self, name, teacher, duration, priority=1, unavailable_timeslots=None):
        self.name = name
        self.teacher = teacher
        self.duration = duration
        self.priority = priority
        self.unavailable_timeslots = unavailable_timeslots if unavailable_timeslots else []

class Schedule:
    def __init__(self, courses, timeslots, rooms):
        self.courses = courses
        self.timeslots = timeslots
        self.rooms = rooms
        # 初始化课程表
        self.schedule = {}
        for timeslot in timeslots:
            for room in rooms:
                self.schedule[(timeslot, room)] = None

    def fitness(self):
        fitness = 0
        course_times = {}
        teacher_assigned = {}

        for course in self.courses:
            teacher_assigned[course.teacher] = 0

        for (timeslot, room), course in self.schedule.items():
            if course:
                # 同一课程同一时间不能安排在多个教室
                if (timeslot, course.name) in course_times:
                    fitness -= 1  # 罚分
                else:
                    course_times[(timeslot, course.name)] = room
                    # 每门课程安排一次，加上优先级作为加分项
                    fitness += course.priority
                    # 统计教师被安排的次数
                    teacher_assigned[course.teacher] += 1

                # 如果教师在该时间段不可用，适应度大幅降低
                if timeslot in course.unavailable_timeslots:
                    fitness -= 5 

        # 教师安排课程的合理性评分
        for teacher, assigned_count in teacher_assigned.items():
            if assigned_count == 0:
                fitness -= 5  # 教师没有被分配到课程
            elif assigned_count > 3:
                fitness -= 2  # 教师被分配了超过3次课程
            else:
                fitness += 1  # 教师被合理分配课程

        return fitness

    def mutate(self):
        keys = list(self.schedule.keys())
        timeslot1, room1 = random.choice(keys)
        timeslot2, room2 = random.choice(keys)
        # 交换两个课程
        self.schedule[(timeslot1, room1)], self.schedule[(timeslot2, room2)] = self.schedule[(timeslot2, room2)], self.schedule[(timeslot1, room1)]

    def crossover(self, other):
        crossover_point = random.randint(0, len(self.schedule) - 1)
        child1_schedule = {}
        child2_schedule = {}

        for i, key in enumerate(self.schedule.keys()):
            if i <= crossover_point:
                child1_schedule[key] = self.schedule[key]
                child2_schedule[key] = other.schedule[key]
            else:
                child1_schedule[key] = other.schedule[key]
                child2_schedule[key] = self.schedule[key]

        child1 = Schedule(self.courses, self.timeslots, self.rooms)
        child2 = Schedule(self.courses, self.timeslots, self.rooms)
        child1.schedule = child1_schedule
        child2.schedule = child2_schedule
        return child1, child2

    def display(self):
        for (timeslot, room), course in self.schedule.items():
            if course:
                print(f"{timeslot} in {room}: {course.name} by {course.teacher} (Priority: {course.priority})")
            else:
                print(f"{timeslot} in {room}: Free")

def genetic_algorithm(courses, timeslots, rooms, population_size=100, fitness_threshold=20):
    population = []
    for _ in range(population_size):
        schedule = Schedule(courses, timeslots, rooms)
        for key in schedule.schedule.keys():
            schedule.schedule[key] = random.choice(courses)
        population.append(schedule)

    generation = 0
    while True:
        # 计算适应度
        population.sort(key=lambda x: x.fitness(), reverse=True)

        # 保留最优的两个个体
        next_generation = []
        for i in range(2):
            next_generation.append(population[i])

        # 选择和交叉
        while len(next_generation) < population_size:
            parent1 = random.choice(population[:5])
            parent2 = random.choice(population[:5])
            child1, child2 = parent1.crossover(parent2)
            next_generation.append(child1)
            next_generation.append(child2)

        # 子代有概率变异
        for schedule in next_generation[2:]:
            if random.random() < 0.1:
                schedule.mutate()

        # 更新种群为下一代
        population = next_generation

        # 获取最优个体
        best_schedule = max(population, key=lambda x: x.fitness())
        best_fitness = best_schedule.fitness()

        # 打印当前代的最优适应度
        generation += 1
        print(f"Generation {generation}: Best Fitness = {best_fitness}")
        best_schedule.display()
        print("-" * 40)

        # 如果达到了适应度要求，退出循环
        if best_fitness >= fitness_threshold:
            break

    return best_fitness, best_schedule

if __name__ == "__main__":
    # 示例课程、时间段和教室
    courses = [
        Course("class1", 'teacher1', 1, priority=3, unavailable_timeslots=["time2", "time4"]),
        Course("class2", "teacher2", 1, priority=2, unavailable_timeslots=["time1"]),
        Course("class3", "teacher3", 1, priority=1),
        Course("class4", "teacher4", 1, priority=3, unavailable_timeslots=["time3"]),
        Course("class5", "teacher5", 1, priority=2),
        Course("class6", "teacher6", 1, priority=1, unavailable_timeslots=["time5", "time6"]),
        Course("class7", "teacher7", 1, priority=3),
        Course("class8", "teacher8", 1, priority=2, unavailable_timeslots=["time2"]),
        Course("class9", "teacher9", 1, priority=1),
        Course("class10", "teacher10", 1, priority=3, unavailable_timeslots=["time1", "time6"]),
        # 添加更多课程
    ]
    timeslots = ["time1", "time2", "time3", "time4", "time5", "time6"]
    rooms = ["Room1", "Room2", "Room3"]

    # 种群大小和适应度阈值
    population_size = 10
    fitness_threshold = 50

    # 运行遗传算法
    best_fitness, best_schedule = genetic_algorithm(courses, timeslots, rooms, population_size, fitness_threshold)
    print("Final Best Schedule:")
    best_schedule.display()
